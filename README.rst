Podium plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/wingraptor/tutor-podium

Usage
-----

::

    tutor plugins enable podium
    Run command the `tutor config save`
